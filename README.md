## potato-chatroom
馬鈴薯聊天室，是一個即時一對一聊天室（非公共聊天室），用戶可透過聊天室與好友進行即時一對一聯繫；每位用戶都擁有獨立的UID，並擁有好友系統。
成為好友前，必須先與其他用戶要求ID，並寄出好友邀請後，等待對方同意後，就可成為好友，並建立獨立聊天室。
